# 🎮 Finn's Arcade

A personal game collection PWA — plays like an app, no App Store needed!

## Games included
- 🎁 Puppet Shooter
- ☠️ Nightmare Shooter  
- 🦊 Mangle vs Stitchwraith
- 🍎 Apple Man Shooter
- 🧁 Mr. Cupcake 3D
- 🧸 Helpy Run!
- 🍏 Apple Catch

## How to publish (GitHub Pages)

1. Go to github.com and sign in as Ancient13
2. Click **New repository** → name it `finns-arcade`
3. Make it **Public**
4. Upload ALL files from this folder (drag and drop)
5. Go to **Settings → Pages**
6. Under Source, select **main branch → / (root)**
7. Click Save
8. Your arcade is live at: `https://ancient13.github.io/finns-arcade`

## How friends install it as an app

**iPhone (Safari):**
1. Open `https://ancient13.github.io/finns-arcade` in Safari
2. Tap the Share button ↑
3. Tap "Add to Home Screen"
4. Tap Add — done! 🎮

**Android (Chrome):**
1. Open the URL in Chrome
2. Tap the banner that says "Add to Home Screen"
   OR tap ⋮ menu → "Install App"
3. Done! 🎮

## Adding new games
Just upload a new `.html` file to the repo, then add a card to `index.html`.
